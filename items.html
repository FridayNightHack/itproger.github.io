Machine Description

Performance Features

The 1.5 mm TDF Pneumatic Folding Machine adopts the 4-cylinder pneumatic system as the power source. Two cylinders are used for bending when the other two are used for pressing. It works stably without power, only needs to connect with the air pump. 1.5mm TDF pneumatic folding machine

The worker can operate the upper mold up down and folding and releasing by controlling the foot pedal. 1.5 mm TDF Pneumatic Folding Machine usually needs two people to operate the pneumatic sheet folding machine. The max folding thickness is 1.5 mm.

There are two types that can be selected. If you only want to bend flat sheets, the pneumatic flat sheet folding machine would be a better one. You have to choose a pneumatic TDF flange sheet folding machine if you need to bend the TDF sheet and you can use it to bend a flat sheet too. 1.5mm TDF pneumatic folding machine

As a piece of efficient equipment for bending TDF flange duct sheet metal, this machine can be customized according to customers’ requirements.
